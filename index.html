    ğŸ’» Usage

    After installation, the Signature Core nodes will be available in your ComfyUI workspace under the "ğŸ”² Signature Nodes" category. Each node is designed to be intuitive and includes proper input validation and error handling.

    Example Workflow

    1. Load an image using ImageFromWeb or ImageFromBase64
    2. Apply transformations using nodes like ImageTranspose or UpscaleImage
    3. Process the image using various filter nodes
    4. Export the result using PlatformOutput or save directly
